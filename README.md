<h1>Documentacion del proyecto</h1>
<h2>Descripcion: </h2>
<p>El proyecto seleccionado es una pagina informativa con objetivo de una pagina de un club de hanball con ubicacion en Marcos Paz, centrada en contenido claro, estructurado y de fácil lectura, sin incluir
  funcionalidades complejas de comercio electrónico. Una página informativa de un club deportivo eficiente centraliza noticias, actividades, aranceles, gestión de socios y multimedia. Permitira a los usuarios
  poder ver fixtures, resultados y proximos partidos asi como tambien ver las redes sociales del equipo</p>
